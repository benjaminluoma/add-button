The plugin adds a title and a rel="nofollow" checkbox to the insert link popup box.

== Description ==
This plugin adds a checkbox to the Insert/Edit Link box for adding `class="btn"` to your links.

This is heavily based on "Title and Nofollow For Links" by Flector: https://wordpress.org/plugins/title-and-nofollow-for-links/

== Installation ==

1. Upload `add-button` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. That's all.
